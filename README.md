# Dragon-bot v1.0.0

بوت تيليجرام بسيط مع نظام نقاط وإحالة.

## المميزات
- نظام نقاط
- نظام إحالة
- أزرار ديناميكية
- نظام أدمن بسيط
- زر تفاصيل البوت
- زر تواصل مع المطور (@ahmaddragon)
- زر إرسال ملاحظات أو أفكار

## التثبيت
1. قم بتثبيت المكتبات: `pip install -r requirements.txt`
2. أضف توكن البوت في config.py
3. شغل البوت: `python main.py`

## الاستخدام
- /start: بدء البوت
- /admin: قائمة الأدمن (للأدمن فقط)

### أوامر الأدمن
- /ban <user_id>: حظر مستخدم
- /unban <user_id>: إلغاء حظر مستخدم
- /add_points <user_id> <points>: إضافة نقاط لمستخدم
- /set_referral_points <points>: تحديث نقاط الإحالة
