# بوت تيليجرام بنظام النقاط والإحالة

هذا البوت تم إصلاحه وتحسينه ليعمل بشكل مستقر على أي استضافة بايثون.

## المميزات التي تم إصلاحها:
- إصلاح جميع أخطاء الاستيراد (Imports).
- التوافق الكامل مع الإصدار الأخير من مكتبة `python-telegram-bot`.
- تحسين استقرار قاعدة البيانات والتعامل مع البيانات.
- إضافة نظام تسجيل أخطاء (Logging) لمتابعة حالة البوت.
- إضافة معالج أخطاء عالمي لمنع توقف البوت عند حدوث خطأ مفاجئ.
- دعم ملفات البيئة `.env` لتأمين المعلومات الحساسة.

## كيفية التشغيل:
1. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
2. (اختياري) قم بإنشاء ملف `.env` بناءً على `.env.example` وضع فيه التوكن الخاص بك.
3. قم بتشغيل البوت:
   ```bash
   python main.py
   ```

## ملاحظات للاستضافة:
- البوت يستخدم `sqlite3` كقاعدة بيانات، وهي ملف محلي يسمى `bot_database.db`.
- تأكد من إعطاء صلاحيات الكتابة للمجلد الذي يحتوي على البوت.
